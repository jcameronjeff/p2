## Ingredients

The styles will be
made up of **Required**, **Selectable**, and **Optional** classes.

### Required

- `.btn` - Class for our un-themed base button.

### Selectable

**variants**

- `.btn-primary` - Solid Button
- `.btn-secondary` - Outline Button
- `.btn-tertiary` - Text Button

<div class='container mb-3'>
  <div className='row'>
    <button className='btn btn-secondary btn-lg m-1'>Large </button>
    <button className='btn btn-secondary btn-md m-1'>Medium </button>
    <button className='btn btn-secondary btn-sm m-1'>Small </button>
  </div>
</div>

### Optional

**sizes**

Button is available in a specific range of sizes (SM - LG) and uses the following class names:

- `.btn-sm` - Small
- `.btn-md` - Medium
- `.btn-lg` - Large

<div class='container my-3'>
  <div className='row'>
    <button className='btn btn-secondary btn-lg m-1'>Large </button>
    <button className='btn btn-secondary btn-md m-1'>Medium </button>
    <button className='btn btn-secondary btn-sm m-1'>Small </button>
  </div>
  <div className='row'>

      <UserIcon />

      <UserIcon />

      <UserIcon />
    </button>
  </div>
</div>

**types**

- `.btn-icon`
- `.btn-aux`
- `.btn-floating`

## Developer Kitchen

Now it's your turn! Select options for the Button component and see how they can
be combined in the final output.

export const argTypes = {
  label: {
    control: {
      type: 'text',
      maxLength: 2,
      className: 'w-12',
    },
  },
  variant: {
    control: {
      type: 'select',
      options: ['primary', 'secondary', 'tertiary'],
    },
  },
  size: {
    control: {
      type: 'select',
      options: ['sm', 'md', 'lg'],
    },
  },
  type: {
    control: {
      type: 'select',
      options: ['default', 'icon', 'aux', 'floating'],
    },
  },
  prepend: {
    label: 'Prepend Icon',
    control: {
      type: 'boolean',
    },
  },
  append: {
    label: 'Append Icon',
    control: { type: 'boolean' },
  },
};

<Kitchen
  args={{
    label: 'Button',
    size: 'md',
    type: 'default',
    variant: 'primary',
  }}
  argTypes={argTypes}
  render={({ label, size, type, variant, prepend, append }) => {
    return (
      <button
        className={clsx(
          'btn',
          `btn-${size}`,
          `btn-${variant}`,
          type !== 'default' && `btn-${type}`
        )}
      >
        {type === 'icon' && <UserIcon className='w-3 h-3' />}
        {type !== 'icon' && (
          <>
            {prepend && <UserIcon className='w-3 h-3' />}
            {label}
            {append && <UserIcon className='w-3 h-3' />}
          </>
        )}
      </button>
    );
  }}
/>
