# Avatar Component Recipe Template

## Definition

  <img src="stories/assets/avatar_48x48.png" />
  <span class="inline-flex text-sm sb-unstyled">An avatar is a representation of an individual user. It can be used as a link to an individual’s account content.</span>

## Content

An avatar should contain a scalable image of the user. If an image is not available, the avatar should display the user’s first and last initials. If neither is available, the avatar should display a generic person icon.

## Ingredients

Avatars are made up of a combination of __elements__ and __styles__.  The styles will be made up of base, themeable, and __configurable__ styles.

### Elements
Avatar elements consist of a `<div>` wrapper, can also be a `<span>`  containing one of the following or all three as needed. 
  - A svg element with path for the icon
  - A background image within the url in the wrapper
  - A span element that contains the initials as text

### Base Styles

 - Base (Wrapper) Styles: `avatar flex items-center justify-center rounded-full`

 - Text Styles: `translate-y-[1px] text-2xl`

 ### Themeable Styles

#### Clickable Styles:
 - Wrapper Clickable Styles: `bg-white border border-blue-700 hover:cursor-pointer`
 - Wrapper Image Styles: `bg-cover ring-1 ring-white ring-inset bg-[url('...')]`
 - Icon Clickable Styles: `text-blue-700`
 - Text Clickable Styles: `text-blue-700`

#### Non-Clickable Styles:
  - Wrapper Non-Clickable Styles: `bg-gray-100`
  - Wrapper Image Styles: `bg-cover bg-[url('...')]`
  - Icon Non-Clickable Styles: `text-black`
  - Text Non-Clickable Styles: `text-black`

### Configurable Style

Avatars are round and contain a scalable graphic or text. The avatar can be xs (24px), sm (32px), md (40px), or lg (48px).

<section class="flex justify-center gap-x-0 gap-y-2 m-auto mb-2 w-max-[600px]">

__Size Styles:__
- `xs`: `w-6 h-6`
- `sm`: `w-8 h-8`
- `md`: `w-10 h-10`
- `lg`: `w-12 h-12`

Nested icon Size Styles tag (svg):
- `xs`: `w-3 h-3`
- `sm`: `w-4 h-4`
- `md`: `w-5 h-5`
- `lg`: `w-6 h-6`
</section>

How to use the styles within the Avatar examples

<section class="gap-4 m-auto mb-2 w-max-[600px]">
  
  <div class="flex justify-center gap-2 sb-story sb-unstyled">
  
    <span class="flex items-center justify-center w-6 h-6 bg-white border border-blue-700 rounded-full avatar hover:cursor-pointer">
      <UserIcon class="w-3 h-3 text-blue-700" alt="Avatar Icon" />
    </span>
    {/* Icon size is half of the div wrapper size */}
    ### Avatar with icon in size xs (24px)
  
  </div>

  Avatar with an image and is clickable uses the following elements: a single `<div>` or a `<span>` and a nested `<UserIcon>`. You will need to import the `<UserIcon>` from the library.
  
  ```sh
  // Typically, you can import from the icons interstate library
  import { UserIcon } from '@prism2/icons-react/interstate';
  ```
  
  You will need to add the following styles to the wrapping element: 
  - `base.styles` : `avatar flex items-center justify-center rounded-full bg-cover`
  - `clickable.styles` : `bg-white border border-blue-700 hover:cursor-pointer`
  - `size.styles` : `w-6 h-6` for xs (24px)
  
  In addition, the `<UserIcon>` styles are: 
  - `clickable.styles` : `text-blue-700` and the `size.styles` : `w-3 h-3`

</section>

```html
<div class="flex items-center justify-center w-6 h-6 bg-white border border-blue-700 rounded-full avatar hover:cursor-pointer">

</div>
```
