## Usage

### Guidance

Represent missing data with a double hyphen “--” using the style for data in the application.

### Rationale

This lets the user know that data should appear here, but is not currently available. 
Leaving the area blank can leave the user wondering if the system is working. 
“n/a” may be confused with “not applicable”. 
“null” is term that may not be widely understood.

## Avoid

- Don't leave the data blank.
- Don't apply a different text style or color.
- Don't use “n/a” or “null”.
