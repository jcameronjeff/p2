## Description

Links allow a user to navigate to another location via descriptive text.

## Ingredients

Links are made up of a combination of __elements__ and __classes__.  The styles will be made up of __Required__ and __Selectable__ .

### Required
  - label: Is used to display the text that is clickable in the link

### Selectable
  - size(s) are available as the following: `link-xxs`, `link-xs`, `link-sm`, `link-md`, `link-lg`, `link-xl`, `link-2xl`, `link-3xl`
  - icon left: Uses the following class in `<img class="link-icon" />` and preceeds the `label`
  - icon right: Uses the following class in `<img class="link-icon" />`and follows after the `label`
  - type [standalone, inline]: The type is used for links that are in the document. By default the inline version can be used in paragraphs or other bodies of naturally flowing text. If the desire is to stand alone, then use `link-stand-alone` class. 

### Optional

There are no optional ingredients.

## Directions
  - Link specs apply to all font sizes from text-xxs to text-3xl
  - Links always use the font-medium weight
    - In body text, designers should use the underline link styles.
    - For links that are not in body text, devlopers should use the link styles, which does not have an underline, but does allow for a left or right icon.  See: Standalone Label below

export const argTypes = {
  tag: {
    label: 'Tag name',
    control: {
      type: 'select',
      options: ['a', 'span', 'div']
    },
  },
  label: {
    control: { type: 'text' },
  },
  size: {
    label: 'Select a Size',
    control: { 
      type: 'select', 
      options: ['xxs', 'xs', 'sm', 'md', 'lg', 'xl', '2xl', '3xl'] 
    },
  },
  iconLeft: {
    left: 'Show left Icon?',
    control: { type: 'boolean' },
  },
  iconRight: {
    left: 'Show right Icon?',
    control: { type: 'boolean' },
  },
  type: {
    label: 'Select the display type',
    control: { 
      type: 'select', 
      options: ['inline', 'stand-alone'],
    },
  },
};

<Kitchen
  args={{ tag: 'a', label: 'link element', type: 'inline' }}
  argTypes={argTypes}
  render={({ label, tag: Tag, type, size = 'lg', iconLeft, iconRight }) => {
    const props = {};
    if (Tag === 'a') {
      props.href = '#'
    } 
    
    return (
      <Tag
        className={clsx(
          Tag !== 'a' && 'link',
          `link-${size}`,
          type !== 'inline' && `link-${type}`
        )}
        {...props}
      >
        {iconLeft && <PhotoIcon className="link-icon" />}
        {label}
        {iconRight && <PhotoIcon className="link-icon" />}
      </Tag>
    )
  }}
/>
