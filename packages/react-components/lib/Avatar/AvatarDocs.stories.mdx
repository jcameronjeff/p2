## Usage

### Common Characteristics

Avatars are round and contain a scalable graphic or text. The avatar can be 24px (xs), 32px (sm), 40px (md), or 48px (lg).

### Content

An avatar should contain a scalable image of the user. If an image is not available, the avatar should display the userâ€™s first and last initials. If neither is available, the avatar should display a generic person icon.
