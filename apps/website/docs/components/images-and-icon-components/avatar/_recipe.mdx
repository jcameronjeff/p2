## Definition

<hr className='my-2' />

## Content

An avatar should contain a scalable image of the user. If an image is not available, the avatar should display the user’s first and last initials. If neither is available, the avatar should display a generic person icon.

<hr className='my-2' />

## Ingredients

Avatars are made up of a combination of **elements** and **styles**. The styles will be made up of base, themeable, and **configurable** styles.

<hr className='my-2' />

### Elements

Avatar elements consist of a `<div>` wrapper, can also be a `<span>` containing one of the following or all three as needed.

- A svg element with path for the icon
- A background image within the url in the wrapper
- A span element that contains the initials as text

<hr className='my-2' />

### Base Styles

- Base (Wrapper) Styles: `avatar flex items-center justify-center rounded-full`

- Text Styles: `translate-y-[1px] text-2xl`

<hr className='my-4' />

### Themeable Styles

<hr className='my-2' />

#### Clickable Styles:

- Wrapper Clickable Styles: `bg-white border border-blue-700 hover:cursor-pointer`
- Wrapper Image Styles: `bg-cover ring-1 ring-white ring-inset bg-[url('@prism2/icons/interstate/user.svg')]`
- Icon Clickable Styles: `text-blue-700`
- Text Clickable Styles: `text-blue-700`

<hr className='my-2' />

#### Non-Clickable Styles:

- Wrapper Non-Clickable Styles: `bg-gray-100`
- Wrapper Image Styles: `bg-cover bg-[url('@prism2/icons/interstate/user.svg')]`
- Icon Non-Clickable Styles: `text-black`
- Text Non-Clickable Styles: `text-black`

<hr className='my-2' />

### Configurable Style

Avatars are round and contain a scalable graphic or text. The avatar can be xs (24px), sm (32px), md (40px), or lg (48px).

<section className="flex justify-center gap-x-0 gap-y-2 m-auto my-4 w-max-[600px]">

**Size Styles:**

- `xs`: `w-6 h-6`
- `sm`: `w-8 h-8`
- `md`: `w-10 h-10`
- `lg`: `w-12 h-12`

Nested icon Size Styles tag (svg):

- `xs`: `w-3 h-3`
- `sm`: `w-4 h-4`
- `md`: `w-5 h-5`
- `lg`: `w-6 h-6`

</section>

export const argTypes = {
  label: {
    control: { type: 'text' },
  },
  size: {
    label: 'Select a Size',
    control: { type: 'select', options: ['xs', 'sm', 'md', 'lg'] },
  },
  clickable: {
    label: 'Toggle Clickable',
    control: { type: 'boolean' },
  },
};

<Kitchen
  args={{ label: 'JC' }}
  argTypes={argTypes}
  render={({ label, clickable, size = 'lg' }) => (
    <div
      className={clsx(
        'avatar',
        `avatar-${size}`,
        clickable && 'avatar-clickable'
      )}
    >
      <span>{label}</span>
    </div>
  )}
/>
